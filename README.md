# Agentex

Agentex is an early prototype for a non-custodial financial orchestration surface for autonomous agents.

This prototype explores:

- playful movable agent avatars
- a natural command surface for owners
- prepared action sheets
- simulation mode
- slide-to-approve approvals
- a bottom Score tray for agent activity

## Run locally

```powershell
npm.cmd start
```

Then open:

```text
http://127.0.0.1:4173
```

You can also open `index.html` directly in a browser.
